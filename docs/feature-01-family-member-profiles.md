# Feature 1: Family Member Profiles

## Scope

Create profile pages for Alex, Amelie, and Ryan with core identifying data and recent health records.

## Acceptance Criteria

- The system lists all family members with name, age, gender, and family role.
- Each member has a dedicated detail view.
- Profiles include health data records, even if the record list is empty.
- The backend exposes a stable API for listing and retrieving profiles.
- The frontend handles loading failure gracefully.
- Seed data is available for Alex, Amelie, and Ryan.

## Out of Scope

- Editing profiles
- Growth charts
- Exercise trend visualizations
- Apple Health sync
- Authentication UI
